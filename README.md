# AutoDNS
AutoDNS es un script híbrido Batch + PowerShell que detecta los servidores DNS públicos más rápidos, evalúa su latencia y aplica los dos mejores al adaptador activo. Compatible con Windows 8.1/10/11 y versiones LTSC, ideal para optimizar conexión, reducir lag en juegos y probar DNS en redes filtradas. Requiere ejecución como administrador.
